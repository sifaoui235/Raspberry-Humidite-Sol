# Raspberry-Humidit--Sol
Mesure du l'humidité de sol avec Raspberry Pi

## Objectif du projet
- Mesurer l’humidité du sol de manière fiable et automatique.
- Faciliter la mise en place d’un système d’irrigation intelligent.
- Fournir un guide pratique pour le câblage et l’installation.

# Contenu
- Raspberry-HumiditéNumerique-Sol.pdf – Documentation complète du système numérique.
- cablage.png – Schéma du câblage pour connecter le capteur au Raspberry Pi.

## Technologies
- Raspberry Pi
- Capteur d’humidité du sol numérique
- Langage Python
